== 👥 그룹 관련 응답 코드 (U02)

> User 와 Group 에 대한 도메인입니다.

|===
| 기능 코드 | 설명

| 00 | 그룹 검색
| 01 | 그룹 내 유저 검색
| 02 | 그룹 생성
| 03 | 그룹 제거
| 04 | 그룹 정보 갱신
| 05 | 유저 → 그룹 가입
| 06 | 유저 → 그룹 탈퇴
| 07 | 그룹 → 유저 강퇴
| 08 | 그룹 → 유저 초대
| 09 | 그룹 초대 코드 생성
| 10 | 그룹 태그 조회
| 12 | 그룹 대표 정보 조회
|===
